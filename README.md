# 📊 Amazon Sales & Consumer Behavior Analysis

## 🎯 Project Overview
This project provides a deep dive into Amazon sales data, focusing on the relationship between product quality (ratings), pricing strategies (discounts), and shipping methods (Prime/FBA).

## 🛠️ Tech Stack
- **Power BI**: Dashboarding and Visualization.
- **Power Query (M Language)**: Data cleaning and conditional coupon extraction.
- **DAX**: Advanced measures for calculating actual discount values and net revenue.

## 🔑 Key Insights
- **The Golden Spot**: Engagement peaks at a **4.3-star rating**.
- **Amazon Choice Paradox**: Non-badge products generated **$140K** vs **$20K** for badged items.
- **Prime Power**: Prime (FBA) sales account for **71.92%** of total volume.

## 📂 Data Model
The project utilizes a relational schema including:
- **Products & Categories**.
- **Student/Instructor Enrollment Schema** (for the database structural part).
